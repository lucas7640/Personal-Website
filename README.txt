LUCAS EDWARD — portfolio site (self-contained build)

Every page has its styling and scripts built in, so there are NO folders to lose.

TO PUBLISH ON GITHUB PAGES
1. Create a new repository (e.g. named  yourusername.github.io  for a root URL).
2. Click  Add file -> Upload files.
3. Select ALL the files in this folder (the .html files AND the .nojekyll file)
   and drag them in together. They all sit at the top level - nothing nested.
4. Commit.
5. Settings -> Pages -> Source -> Deploy from a branch -> main / root -> Save.
6. Wait ~1 minute, then visit https://yourusername.github.io/

TIP: make sure .nojekyll is included (it may be hidden in Finder - press
Cmd+Shift+. to show hidden files).
